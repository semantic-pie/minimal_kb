В тех случаях, когда некоторая сущность связана с другими сущностями связкой какого-либо отношения и известны роли компонентов связки, а также идентификаторы хотя бы некоторых этих компонентов, то идентификаторы других компонентов могут быть построены путем конкатенации идентификатора соответствующего ролевого отношения и идентификатора другого компонента этой же связки, взятого в скобки.

Например:
подмножество'(s)
sin'(x)
log'(x)
