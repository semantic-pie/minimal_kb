максимальный абстрактный sc-агент представляет собой неатомарный абстрактный sc-агент, являющийся результатом объединения всех абстрактных sc-агентов, входящих в состав какой-либо конкретной ostis-системы в один. Другими словами под максимальный абстрактный sc-агент понимается коллектив всех sc-агентов, входящих в состав заданной ostis-системы.
