мощность – это параметр, элементами которого являются множества, имеющие одинаковое количество элементов. Значением данного параметра является числовая величина, задающая количество элементов, входящих в данный класс множеств, то есть по сути, количество позитивных sc-дуг принадлежности, выходящих из данного множества.

Для двух множеств, имеющих одинаковую мощность, существует взаимно-однозначное соответствие между ними (между множествами вхождений элементов в эти множества – на случай мультимножеств).
