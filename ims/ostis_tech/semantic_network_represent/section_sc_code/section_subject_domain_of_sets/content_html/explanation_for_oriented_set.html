ориентированное множество – это множество, представляющее собой упорядоченный набор элементов, т.е. такое множество, порядок элементов в котором имеет значение. Пары принадлежности элементов ориентированному множеству могут дополнительно принадлежать каким-либо ролевым отношениям, при этом, в рамках каждого ориентированного множества должен существовать хотя бы один элемент, роль которого дополнительно уточнена ролевым отношением.
