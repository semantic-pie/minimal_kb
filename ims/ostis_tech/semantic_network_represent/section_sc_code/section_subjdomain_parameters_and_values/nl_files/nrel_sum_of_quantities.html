сумма величин* — это арифметическая операция на величинах, аналогичная арифметической операции сумма* для чисел.

Первым компонентом связки отношения сумма величин* является подмножество некоторого количественного параметра (слагаемые величины), содержащее два или более элемента, вторым компонентом – элемент этого же количественного параметра, значение которого в любой единице измерения* является результатом сложения значений всех слагаемых величин в той же единице измерения*. При несовпадении единиц измерения слагаемых величин необходимо воспользоваться соотношениями между единицами измерения, которые задаются при помощи операций произведение величин* и возведение величин в степень*.
