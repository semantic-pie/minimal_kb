равенство величин* — это арифметическая операция на величинах, аналогичная арифметической операции равенство* для чисел.

Отношение равенство величин* носит исключительно дидактический характер, и явно не указывается, поскольку связывает попарно все элементы одной и той же величины каждого количественного параметра.
