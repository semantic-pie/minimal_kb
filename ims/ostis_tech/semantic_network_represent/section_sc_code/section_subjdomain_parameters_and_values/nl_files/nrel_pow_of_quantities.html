возведение величин в степень* — это арифметическая операция на величинах, аналогичная арифметической операции возведение в степень* для чисел.

Первым компонентом связки отношения возведение величин в степень* является ориентированная пара, первым компонентом которой является величина количественного параметра (основание степени), вторым – число (показатель степени); Вторым компонентом связки отношения возведение величин в степень* является величина количественного параметра (результат возведения в степень).
