Действия класса конъюнкция предшествующих действий используются в тех случаях, когда выполнение некоторого действия должно начаться только после того, как будут выполнены все предшествующие действия, а не только одно из них. После того, как все предшествующие действия выполнены, инициируются действия, следующие за конъюнкцией предшествующих действий.
