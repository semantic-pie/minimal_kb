В случае процедурной формулировки задачи, в формулировке задачи явно указываются аргументы соответствующего задаче действия, и в частности, вводится семантическая типология действий. При этом явно не уточняется, что должно быть результатом выполнения данного действия. Заметим, что, при необходимости, процедурная формулировка задачи может быть сведена к декларативной формулировке задачи путем трансляции на основе некоторого правила, например определения класса действия через более общий класс.
