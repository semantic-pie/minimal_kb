Отношение следующий оператор* используется для указания scp-оператора, который должен выполниться следующим после выполнения текущего scp-оператора, в случае, если выполнение текущего оператора завершилось без ошибок, и нет никаких дополнительных условий успешности выполнения, зависящих от конкретного класса оператора. Переход в таком случае можно считать безусловным. Данное ролевое отношение используется в большинстве scp-операторов, за исключением scp-операторов ассоциативного поиска sc-конструкций и scp-операторов проверки условий. Допускается использование нескольких выходящих пар данного отношения для одного и того же оператора в случае, если после завершения выполнения текущего оператора предполагается выполнение нескольких операторов параллельно.
