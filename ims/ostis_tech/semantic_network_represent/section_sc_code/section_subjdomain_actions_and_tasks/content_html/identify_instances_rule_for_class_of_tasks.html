Конкретные классы задач в рамках Русского языка именуются по следующим правилам:

  • в начале идентификатора пишется слово задача и ставится точка;
  • • далее со строчной буквы идет либо содержащее глагол совершенного вида в инфинитиве описание сути того, что требуется получить в результате решения данного класса задач, либо вопросительное предложение, являющееся спецификацией запрашиваемой (ответной) информации.

Например:

задача. сформировать полную семантическую окрестность указываемой сущности

задача. верифицировать заданную sc-структуру

Допускается использовать менее строгие идентификаторы, которые, однако, обязаны оперировать словом задача и достаточно четко специфицировать суть задач описываемого класса.

Например:

задача на установление значения величины

задача на доказательство
