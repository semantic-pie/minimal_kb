Отношение следующий оператор при безуспешном выполнении* используется для указания scp-оператора, который должен выполниться следующим после выполнения текущего scp-оператора, в случае, если выполнение текущего оператора завершилось без ошибок, однако не выполнено некоторое дополнительное условие, зависящее от конкретного класса оператора (например попытка поиска заданной конструкции оказалась безуспешной при заданных параметрах поиска). Допускается использование нескольких выходящих пар данного отношения для одного и того же оператора в случае, если после завершения выполнения текущего оператора предполагается выполнение нескольких операторов параллельно.
