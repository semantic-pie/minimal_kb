Экземпляры класса действий, в рамках Русского языка именуются по следующим правилам:

  • в начале идентификатора пишется слово Действие и ставится точка;
  • далее с прописной буквы идет либо содержащее глагол совершенного вида в инфинитиве описание сути того, что требуется получить в результате выполнения действия, либо вопросительное предложение, являющееся спецификацией запрашиваемой (ответной) информации.

Например:

Действие. Сформировать полную семантическую окрестность понятия треугольник

Действие. Верифицировать Раздел. Предметная область sc-элементов
