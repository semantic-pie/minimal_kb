Связки отношения заказчик* связывают sc-элементы, обозначающие действие и sc-элементы, обозначающие субъекта, который «заинтересован» в выполнении данного действия и, как правило, инициирует его выполнение. Данное отношение может быть использовано при указании того, кто поставил проектную задачу по развитию баз знаний.

Первым компонентом связок отношения заказчик* является знак действия, вторым - знак субъекта-заказчика.
