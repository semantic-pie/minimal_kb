Экземпляры класса задач в рамках Русского языка именуются по следующим правилам:

  • в начале идентификатора пишется слово Задача и ставится точка;
  • далее с прописной буквы идет либо содержащее глагол совершенного вида в инфинитиве описание сути того, что требуется получить в результате решения данной задачи, либо вопросительное предложение, являющееся спецификацией запрашиваемой (ответной) информации.

Например:

Задача. Сформировать полную семантическую окрестность понятия треугольник

Задача. Верифицировать Раздел. Предметная область sc-элементов
