Во множество действий, выполненных с ошибкой, попадают действия, выполнение которых не было успешно завершено с точки зрения субъекта, осуществлявшего их выполнение, по причине возникновения какой-либо ошибки, например, некорректности спецификации данного действия или нарушения целостности sc-памяти каким-либо субъектом (в случае действия в sc-памяти).
