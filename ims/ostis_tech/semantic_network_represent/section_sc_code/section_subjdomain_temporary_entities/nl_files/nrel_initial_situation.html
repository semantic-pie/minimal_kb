Связки отношения начальная ситуация* связывают некоторый процесс и некоторую ситуацию, являющуюся начальной для этого процесса, и, как правило, изменяемой в течение выполнения этого процесса.

Первым компонентом каждой связки отношения начальная ситуация* является знак процесса, вторым – знак начальной ситуации.
