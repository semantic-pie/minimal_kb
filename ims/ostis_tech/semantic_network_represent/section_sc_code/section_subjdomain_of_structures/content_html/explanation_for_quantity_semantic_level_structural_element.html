количество семантических уровней элементов структуры – параметр, каждый элемент которого представляет собой класс структур, у которых совпадает максимальный среди семантических уровней элементов этих структур.

Значением данного параметра является число, совпадающее с указанным максимальным семантическим уровнем элементов.
