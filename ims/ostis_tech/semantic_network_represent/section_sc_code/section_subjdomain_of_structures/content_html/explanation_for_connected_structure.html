Структуре, представленной в SC-коде, поставим в соответствие орграф, вершинами которого являются sc-элементы, а дугами – пары инцидентности, связывающие sc-коннекторы с инцидентными им sc-элементами, которые являются компонентами указанных sc-коннекторов.

Если полученный таким способом орграф является связным орграфом, то исходную структуру будем считать связной структурой.
