Каждый элемент параметра кратность существования представляет собой класс логических , для которых при интерпретации на соответствующей предметной области существует ограниченное общее для всех таких формул число комбинаций значений переменных, при которых указанные формулы являются истинными в рамках соответствующей формальной теории.

Таким образом, измерением* каждого значения кратность существования является некоторое число, задающее количество таких комбинаций.
